## How to run

```bash
# clone project to local
git clone https://github.com/Geek-James/ddBuy.git

# go to project catalog
cd ddbuy

# install node package
npm install

# run serve
npm run serve

Browser access http://localhost:8080
```

## Release

```bash
# Build production environment
npm run build
```
